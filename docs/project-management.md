# Gestión del Proyecto - Page & Frame

Para el desarrollo de esta aplicación, estamos utilizando una metodología **Kanban** mediante un tablero en Trello.

## Metodología
El flujo de trabajo se divide en 5 estados:
1. **Backlog:** Ideas y funcionalidades futuras.
2. **To Do:** Tareas priorizadas para la fase actual.
3. **In Progress:** Tareas que se están programando en este momento.
4. **Review:** Pruebas de funcionamiento y revisión de código.
5. **Done:** Funcionalidades completadas y subidas al repositorio.
