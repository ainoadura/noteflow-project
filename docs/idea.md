# Page & Frame - My Media Universe

## 1. El Problema
La dificultad de organizar y compartir recomendaciones de libros, películas y series que suelen quedar perdidas en chats de WhatsApp o notas genéricas.

## 2. Usuario Objetivo
Entusiastas de la cultura que desean un registro visual, puntuado y organizado de sus consumos mediáticos para consulta personal o para compartir con amigos.

## 3. Funcionalidades Principales (MVP)
- **🎬 Tres tipos de contenido (Firma Visual Semántica):**
    - Películas (Rojo Cinematográfico - `#FF4B4B`): Título, director, duración, puntuación y reseña.
    - Series (Morado Streaming - `#8B5CF6`): Título, director, nº capítulos, puntuación y reseña.
    - Libros (Verde Esmeralda Editorial - `#10B981`): Título, autor, nº páginas, puntuación y reseña.
- **Gestión de Listas:** Creación de listas personalizadas (mezclando tipos) y eliminación de las predeterminadas.
- **Navegación:** Página principal con listas y vista detalle para cada post.
- **Ordenación:** Filtro por puntuación (1 a 5 estrellas).
- **Historial (Aside):** Acceso cronológico a los posts creados.

## 4. Funcionalidades Opcionales (Futuro)
- Compartir listas mediante links.
- Buscador avanzado por tipo de contenido.
- Sincronización en la nube para amigos.
